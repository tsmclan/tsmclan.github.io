How to manually git clone repo via https

If you have generated your site, but for some reasons, fail to publish it, and/or you hope to use git manually to push it to Github, you can follow following steps:

Step 1: cd /Users/rchen/.config/joplin-desktop/plugin-data/ylc395.pagesPublisher/output

you can get the output-dir in "Generated Successfully" modal dialogue and then cd <output-dir>

Step 2: git init --separate-git-dir ../tsmclan

git ends with a message "Initialized empty Git repository in /Users/rchen/.config/joplin-desktop/plugin-data/ylc395.pagesPublisher/tsmclan/"
